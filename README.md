# Intrusion Detection System (IDS) Using Machine Learning
The focus of this project, led by Dr. Nhut Nguyen, is to develop an IDS using the open-source [IDS-ML](https://www.sciencedirect.com/science/article/pii/S2665963822001300) framework. This system will have UI capabilities to modify parameters and analyze validation results
